# Rules

1. **무조건 대답은 한국말**
   - 모든 답변과 커뮤니케이션은 한국어로 진행한다.

2. **할 일들을 tasks.md에 나열 (한국말로)**
   - 진행할 작업 내역을 tasks.md 파일에 한국어로 기록하고 관리한다.

3. **무슨 일을 하기 전엔 나에게 허락받고 수행하기**
   - 작업을 시작하기 전에 사용자에게 계획을 알리고 승인을 받은 후 진행한다.

4. **docs/implementation_plan.md**
   - 해당 파일은 '항상 누적'해서 할일을 적고, 한국어로 기록한다.

5. **docs/tasks.md**
   - 해당 파일은 '항상 누적'해서 할일을 적고, 한국어로 기록한다.
