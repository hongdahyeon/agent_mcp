# 라우터 파일 정리

## 라우터란?
- @router.get(...) (데코레이터)
- FastAPI에게 **누군가 이 주소(URL)로 요청을 보내면, 아래 함수를 실행해줘!** 라고 알려주는 역할 수행
- 웹 브라우저나 FE에서 호출할 수 있는 **API 엔드포인트(관문)**를 정의하는 파일


## 파일 정보
1. auth.py
- 로그인, 로그 관련 라우터 파일

2. email.py
- 이메일 관련 라우터 파일

3. execution.py
- 실행 관련 라우터 파일
- Proxy 실행 엔드포인트
- 실행(Proxy) 전용 관문을 따로 분리

4. files.py
- 파일 관련 라우터 파일

5. mcp.py
- mcp 관련 라우터 파일

6. openapi.py
- openapi 관련 라우터 파일
- OpenAPI관리(등록/수정/삭제/통계/관련 설정) 기능에만 집중

7. system.py
- 시스템 관련 라우터 파일

8. users.py
- 유저 관련 라우터 파일
